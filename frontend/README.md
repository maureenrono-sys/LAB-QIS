# Lab QIS Professional (ISO 15189:2022)

## 1. Introduction
Lab QIS is a quality-focused digital platform designed to transform routine laboratory data into actionable quality insights. It operates as a "Quality Intelligence Layer" agnostic to the Laboratory Information System (LIS).

## 2. Key Features
* **Executive Quality Dashboard**: Real-time overview of TAT, critical alerts, and SLIPTA star levels.
* **Passive Data Ingestion**: Supports CSV uploads for Analyzer QC and LIS summary imports to minimize staff burden.
* **Non-Conformity & CAPA**: Structured tracking of errors and corrective actions aligned with Clause 8.7 of ISO 15189.
* **Risk Management**: Aggregated indicators to generate departmental risk profiles.

## 3. Technology Stack
* **Frontend**: HTML5, Bootstrap 5, Chart.js, JavaScript (ES6+)
* **Backend**: Node.js, Express
* **Database**: (e.g., MySQL / PostgreSQL / MongoDB)

## 4. Setup and Installation
1. Clone the repository: `git clone <your-repository-link>`
2. Install dependencies: `npm install`
3. Configure your `.env` file (see `.env.example`).
4. Start the server: `npm start`

## 5. Compliance Alignment
Designed to meet the requirements of:
* **ISO 15189:2022** (Clauses 4.14, 7.3, 8.4, 8.7)
* **SLIPTA Checklist**
